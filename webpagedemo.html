How to Create a Github Web Page

  • 1. First you will access github in the browser, create a username and a password, and log in.
  • 2.After logging in go to repositories and create a new repository.
  • 3.Give the repository a name ("name".github.io).
  • 4.Under quick setup options, select "Upload existing file" That action will open a window for you to drag and drop files.
  • 5.Using VS code create or select file of your choice under the "name".github.io folder.
  • 6.Under your computers local files drag and drop files like the current one into the repository drag and drop window.
  • 7. Select commit changes at the bottom of the screen.
  • 8. Congratulations. You will now be able to log your github.io web page in a browser to access
